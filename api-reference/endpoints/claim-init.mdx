---
title: 'GET /claim/init'
description: 'Initialize claim process'
api: 'GET https://api.pow.cards/api/v1/claim/init'
---

Initialize a claim process by generating a sign-in message.

### Query Parameters

<ParamField query="publicKey" type="string" required>
  The Solana wallet public key
</ParamField>

}
